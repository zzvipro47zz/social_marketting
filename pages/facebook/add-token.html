Thêm token vào hệ thống

- Nhập danh sách token cần đưa vào hệ thống.

- Hệ thống sẽ tự động phân loại token theo 3 loại.

Loại 1: Những token nào đang hoạt động và không có fanpage.

Loại 2: Những token nào đang hoạt động và có fanpage.

Loại 3: Những token nào đang hoạt động và là fanpage.

- Sau khi phân loại token xong, bạn cần nhấn button Lưu tất cả để lưu.
